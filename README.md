## Installation
Currently the SORTEEDCQC package is not on CRAN, but you can install the development version from GitHub using the devtools package:

```{r}
install.packages("devtools")
devtools::install_github("EIvimeyCook/SORTEEDCQC")
library(SORTEEDCQC)
```

## Running SORTEE.DCQC
The only function metRscreen is SORTEEDCQC().

```{r}
library(SORTEEDCQC)
SORTEEDCQC()
```
